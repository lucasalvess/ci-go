# CI GoMath

Simple calculator with golang. Made for study continuous integration with github actions.

## Stack
- Golang

### How to run

**With Docker:**

Build:
```sh
$ docker build . -t "go-math"
```

Run:
```sh
 docker run go-math
```

**With Golang:**


Run:
```sh
 $ go run math.go
```

## Running tests

Run:
```sh
 $ go test
```
